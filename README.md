# 📊 HR Employee Analysis using SQL Server

## 📌 Project Overview
This project focuses on analyzing HR employee data using SQL Server.  
It includes multiple tables like Employee, Department, Pay History, and Shift.

## 🗂️ Tables Used
- HumanResources.Employee
- HumanResources.EmployeePayHistory
- HumanResources.EmployeeDepartmentHistory
- HumanResources.Department
- HumanResources.Shift

## 🔍 Views Used
- dbo.vw_new
- dbo.vw_emp1
- dbo.vw_emp2

## 📈 Key Analysis
- Total Employees
- Gender Distribution
- Marital Status Analysis
- Department-wise Employees
- Shift-wise Employees
- Salary Analysis
- Hiring Trends
- Employee Age Calculation
- Salary Growth

## 🚀 How to Run
1. Open SQL Server Management Studio (SSMS)
2. Connect to database
3. Run SQL scripts from SQL_Scripts folder

## 📷 Output
Screenshots of query results are stored in Outputs folder.

## 🎯 Conclusion
This project demonstrates SQL skills including joins, aggregations, grouping, and data analysis.
